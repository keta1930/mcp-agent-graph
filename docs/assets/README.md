# Documentation Assets

This directory contains static assets for the documentation.

## Required Files

Please add the following files to this directory:

- **logo.png**: Logo for the documentation header (recommended size: 128x128px)
- **favicon.png**: Favicon for the documentation site (recommended size: 32x32px)

## Image Guidelines

- Use PNG format with transparency
- Follow the earth-tone color palette from the style guide
- Keep file sizes optimized for web
- Logo should be clear and legible at small sizes

## Color Palette Reference

When creating logos and graphics, use these colors from the style guide:

- Primary: `#b85845` (rust red)
- Secondary: `#8b7355` (earth brown)
- Accent: `#a0826d` (clay)
- Background: `#faf8f5` (cream)
