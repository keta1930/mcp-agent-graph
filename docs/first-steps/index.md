# First Steps

Get started with MAG by following these essential guides.

## Overview

This section covers everything you need to know to get MAG up and running:

1. **Introduction**: Learn about MAG's architecture and key concepts
2. **Installation**: Install MAG on your system
3. **Quickstart**: Run your first agent and workflow

## Prerequisites

Before you begin, ensure you have:

- Python 3.11 or higher
- Node.js 18 or higher (for frontend)
- MongoDB (for data storage)
- Docker (optional, for containerized deployment)

## What You'll Learn

- MAG's core concepts and architecture
- How to install and configure MAG
- Creating your first agent
- Running a simple workflow
- Basic configuration options
