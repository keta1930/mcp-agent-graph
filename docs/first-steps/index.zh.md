# 快速开始

按照这些基础指南开始使用 MAG。

## 概览

本节涵盖了启动和运行 MAG 所需的所有知识：

1. **介绍**: 了解 MAG 的架构和核心概念
2. **安装**: 在你的系统上安装 MAG
3. **快速入门**: 运行你的第一个智能体和工作流

## 前置要求

在开始之前，请确保你已安装：

- Python 3.11 或更高版本
- Node.js 18 或更高版本（用于前端）
- MongoDB（用于数据存储）
- Docker（可选，用于容器化部署）

## 你将学到什么

- MAG 的核心概念和架构
- 如何安装和配置 MAG
- 创建你的第一个智能体
- 运行简单的工作流
- 基本配置选项
