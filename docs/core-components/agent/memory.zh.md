# Agent 记忆

赋予 Agent 个人知识库，随时间学习和改进。

## 什么是 Agent 记忆

Agent 记忆使 Agent 能够独立于共享用户偏好存储和回忆自己的知识。

| 记忆类型 | 所有者 | 访问权限 | 用途 |
|---------|-------|---------|------|
| **用户记忆** | 您 | 所有 Agent | 您的偏好和需求 |
| **Agent 记忆** | 每个 Agent | 仅该 Agent | Agent 学习的模式和经验 |

## Agent 如何使用记忆

Agent 在对话期间自主管理记忆：

**任务前：** 搜索记忆查找相关模式

**执行中：** 添加学习内容并更新知识

**完成后：** 存储成功的方法

## 核心优势

| 优势 | 结果 |
|-----|------|
| **持久学习** | 知识跨对话积累 |
| **个性化服务** | Agent 记住您的风格和偏好 |
| **自主改进** | Agent 从经验中学习，无需手动输入 |
| **上下文隔离** | 每个 Agent 构建领域特定专业知识 |

## 管理记忆

### 通过记忆管理器

从工作空间打开记忆管理器，查看和管理任何 Agent 的知识：

- 查看分类和条目
- 添加、编辑或删除记忆
- 导出为 JSON、Markdown、TXT 或 YAML
- 通过 AI 解析导入内容

### 通过记忆工具

Agent 在对话中使用内置记忆工具：

- `list_memory_categories` - 发现现有知识
- `get_memory` - 检索特定记忆
- `add_memory` - 存储新学习
- `update_memory` - 完善现有知识
- `search_memory_with_agent` - 在隔离上下文中 AI 驱动搜索

## 常见使用场景

| Agent 类型 | 记忆策略 | 结果 |
|-----------|---------|------|
| 代码审查 Agent | 在 `code_standards` 中存储风格规则 | 一致的审查 |
| 支持 Agent | 在 `faq_answers` 中保存解决方案 | 更快响应 |
| 写作 Agent | 在 `brand_voice` 中记住语气 | 统一风格 |
