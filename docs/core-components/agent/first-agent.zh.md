# 构建您的第一个 Agent

## 准备工作

创建智能体前，确保您已：

- 至少注册了 **一个模型** （参见[模型注册](../model/register.zh.md)）
- 可访问 **工作台** 界面

## 创建 Agent

### 步骤 1：打开 Agent 管理

从工作台侧边栏进入 **Agent 管理**。

### 步骤 2：配置基本信息

点击 **创建 Agent**，填写必填字段：

| 字段 | 说明 | 示例 |
|------|------|------|
| **名称** | 唯一标识符（不含 `/`、`\`、`.`） | `code-reviewer` |
| **能力卡片** | 简要描述智能体功能 | `审查代码的质量、安全性和最佳实践` |
| **分类** | 用于组织分类 | `编程` |
| **标签** | 用于搜索和发现（可选） | `代码`、`审查`、`安全` |

### 步骤 3：选择模型

选择驱动智能体的大语言模型：

- **Claude Sonnet 4.5** - 代码与推理
- **GPT-5** - 通用性能
- **DeepSeek V3** - 快速且经济

模型负责所有决策和工具使用。

### 步骤 4：添加系统指令（可选）

定义自定义行为：

```
您是一位精通 Python、JavaScript 和安全的高级代码审查员。
审查代码时：
- 检查常见安全漏洞
- 建议性能改进
- 突出最佳实践违规
- 提供具体改进示例
```

留空则使用默认助手行为。

### 步骤 5：配置工具（可选）

选择智能体可使用的工具。前往以下页面了解更多：

- **[MCP 服务器](../mcp/index.zh.md)** - 连接外部服务和能力
- **[系统工具](../tools/index.zh.md)** - 使用内置功能

### 步骤 6：设置最大动作数

配置迭代上限（1-200）：

- **10-20** - 简单问答智能体
- **30-50** - 通用智能体（默认：50）
- **100+** - 复杂多步骤工作流

### 步骤 7：创建

点击 **提交** 保存您的智能体。

## 运行 Agent

### 从聊天界面

1. 从工作台打开 **聊天** 页面
2. 点击 **Agent 选择器** 下拉框
3. 选择您新创建的智能体
4. 输入消息并发送

智能体将使用配置的模型和工具处理您的请求。

## 常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 智能体无法运行 | 未选择模型 | 确保已配置模型 |
| 工具不工作 | MCP 服务器断连 | 在 MCP 管理中重新连接 |
| 一次回复后停止 | 简单问题的正常行为 | 尝试多步骤请求 |
| 达到最大动作数 | 任务复杂或死循环 | 增加 max_actions 或简化任务 |
