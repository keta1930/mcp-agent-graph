# 注册您的模型

配置 LLM 模型，在对话、Agent 工作流等场景中使用。

## 快速开始

从工作空间侧边栏进入**模型管理**页面，注册新模型。

## 必填配置

| 字段 | 说明 | 示例 |
|------|------|------|
| 模型名称 | 模型显示名称 | `claude-sonnet-4-5` |
| 提供商 | 模型提供商(当前支持 OpenAI 兼容) | `openai` |
| 模型类型 | 模型类型 | `llm`(语言模型) |
| 基础 URL | API 端点地址 | `https://api.anthropic.com/v1/` |
| API 密钥 | 认证密钥 | `sk-ant-api...` |
| 模型标识符 | 提供商的模型 ID | `claude-sonnet-4-5-20250929` |

## 高级配置

可选参数，用于调整模型行为：

### 生成控制

| 参数 | 范围 | 说明 |
|------|------|------|
| Temperature | 0.0 - 2.0 | 输出随机性(越高越有创意) |
| Max Tokens | ≥ 1 | 最大响应长度 |
| Max Completion Tokens | ≥ 1 | 最大完成令牌数 |
| Top P | 0.0 - 1.0 | 核采样阈值 |

### 响应调优

| 参数 | 范围 | 说明 |
|------|------|------|
| Frequency Penalty | -2.0 - 2.0 | 减少重复(正值 = 更少重复) |
| Presence Penalty | -2.0 - 2.0 | 鼓励话题多样性(正值 = 更多样) |
| Stop Sequence | 字符串或列表 | 停止生成的标记序列 |
| Response Count (n) | 1 - 10 | 生成的候选响应数量 |

### 调试参数

| 参数 | 范围 | 说明 |
|------|------|------|
| Seed | 整数 | 使用相同种子实现确定性生成 |
| Logprobs | 布尔值 | 返回令牌的对数概率 |
| Top Logprobs | 0 - 20 | 返回最可能的令牌候选数量 |

### 连接配置

| 参数 | 说明 |
|------|------|
| Timeout | 请求超时时间(秒) |
| Extra Body | 特定提供商的自定义 JSON 参数 |

## 常用配置

### Claude Sonnet 4.5

```
提供商: openai
基础 URL: https://api.anthropic.com/v1/
模型: claude-sonnet-4-5-20250929
```

### GPT-5

```
提供商: openai
基础 URL: https://api.openai.com/v1/
模型: gpt-5
```

### Gemini 2.5 Flash

```
提供商: openai
基础 URL: https://generativelanguage.googleapis.com/v1beta/
模型: gemini-2.5-flash-latest
```

## 管理模型

### 编辑模型

点击模型卡片上的**编辑**按钮更新配置。编辑时 API 密钥留空可保持原有密钥。

### 删除模型

点击**删除**移除模型。正在被 Agent 或工作流使用的模型无法删除。

### 标题生成模型

设置用于自动生成对话标题的默认模型：

1. 点击页面顶部的**标题模型设置**
2. 选择首选模型
3. 保存

该模型将用于为新对话生成描述性标题。

## 最佳实践

**API 密钥**：安全存储密钥。平台创建后不会显示完整密钥。

**命名规范**：使用清晰的名称如 `gpt-4-turbo-preview`，而非 `model-1`。

**测试验证**：创建测试对话验证模型配置，再用于工作流。

**默认参数**：从不设置高级参数开始，根据使用场景逐步调整。
