# 创建邀请码

生成并管理新成员注册邀请码。

---

## 访问权限

仅管理员可创建和管理邀请码。访问管理后台：

在浏览器中直接访问 `/admin` 页面（例如：`https://agent-graph.com/admin`）

管理员需要登录后才能访问邀请码管理。

---

## 功能概述

邀请码控制谁可以注册平台。每个邀请码包含：
- **唯一码**：自动生成格式为 `TEAM-XXXXXX`
- **使用限制**：可选的注册次数限制
- **状态**：激活或停用
- **描述**：可选的备注说明

---

## 创建邀请码

点击邀请码表格右下角的**生成新邀请码**按钮。

### 配置项

| 字段 | 必填 | 说明 |
|------|------|------|
| 邀请码描述 | 可选 | 用于标识此邀请码用途的备注 |
| 使用次数限制 | 可选 | 限制注册次数（留空表示无限制） |

点击**生成**创建邀请码。

---

## 管理邀请码

### 邀请码信息

在表格中查看所有邀请码：

| 列名 | 说明 |
|------|------|
| 邀请码 | 邀请码本身（点击复制图标可复制） |
| 状态 | 激活或停用 |
| 使用情况 | 当前使用次数 / 限制次数（∞ 表示无限制） |
| 创建者 | 生成此邀请码的管理员 |
| 创建时间 | 邀请码创建日期 |
| 过期时间 | 过期日期（永久表示不过期） |

### 切换状态

点击**停用**可阻止激活状态的邀请码接受新注册。

点击**激活**可重新启用已停用的邀请码。

状态修改立即生效。
