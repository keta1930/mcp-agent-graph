# 创建和编辑 Prompt

在 Prompt 中心管理您的提示词模板。

## 创建 Prompt

在主界面点击"创建 Prompt"按钮打开创建窗口。

**必填信息：**

| 字段 | 说明 | 规则 |
|------|------|------|
| **名称** | Prompt 的唯一标识 | 不超过 100 字符，不含 `/ \ : * ? " < > \|` |
| **分类** | 组织管理的分类标签 | 仅英文字母、数字、连字符和下划线 |
| **内容** | Prompt 的具体文本 | 必填，支持 Markdown 格式 |

**创建流程：**

```
填写名称和分类 → 编写 Prompt 内容 → 点击创建 → 保存成功
```

**名称重复处理：** 如果名称已存在，系统会提示选择其他名称。

## 编辑 Prompt

点击 Prompt 卡片上的编辑图标打开编辑窗口。

**可编辑内容：**

| 字段 | 是否可修改 |
|------|-----------|
| **名称** | ❌ 不可修改（作为唯一标识） |
| **分类** | ✅ 可修改 |
| **内容** | ✅ 可修改 |

**编辑流程：**

```
选择 Prompt → 点击编辑 → 修改分类或内容 → 保存更新
```

## 删除 Prompt

点击 Prompt 卡片上的删除图标触发删除确认。

**删除流程：**

```
点击删除图标 → 确认删除 → 永久删除
```

⚠️ **删除不可恢复：** 删除操作无法撤销，请谨慎操作。

## 导入 Prompt

通过导入功能从本地文件创建 Prompt。

**支持格式：**

| 格式 | 说明 |
|------|------|
| **Markdown (.md)** | 标准 Markdown 文件 |
| **文本 (.txt)** | 纯文本文件 |

**导入流程：**

```
点击导入 → 选择文件 → 填写名称和分类 → 确认导入
```

**导入要求：**

- 文件编码必须为 UTF-8
- 文件内容将作为 Prompt 内容
- 名称和分类仍需手动指定

## 导出 Prompt

从 Prompt 列表选择一个或多个 Prompt 进行导出。

**导出格式：**

| 格式 | 说明 |
|------|------|
| **ZIP 压缩包** | 包含所有选中 Prompt 的 Markdown 文件 |
| **文件命名** | 使用 Prompt 名称作为文件名 |

**导出流程：**

```
选择 Prompt → 点击导出 → 下载 ZIP 文件
```

**文件结构示例：**

```
prompts_export.zip
├── code-review.md
├── summarize.md
└── translate.md
```

## 分类管理

Prompt 自动按分类分组展示。

**分类规则：**

| 规则 | 说明 |
|------|------|
| **自动分组** | 相同分类的 Prompt 显示在同一分组 |
| **未分类** | 空分类的 Prompt 归入"未分类" |
| **分类统计** | 顶部显示分类总数 |

**分类显示：**

每个分类以独立卡片组形式展示，包含该分类下的所有 Prompt。

## 搜索过滤

使用搜索框快速定位 Prompt。

**搜索规则：**

| 搜索方式 | 说明 |
|---------|------|
| **名称匹配** | 按 Prompt 名称模糊搜索 |
| **实时过滤** | 输入即刻显示结果 |
| **大小写不敏感** | 忽略大小写差异 |

**搜索流程：**

```
输入关键词 → 实时过滤列表 → 显示匹配结果
```
