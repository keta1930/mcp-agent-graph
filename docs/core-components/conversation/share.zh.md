# 对话分享

通过公开链接与他人分享您的对话。

## 什么是对话分享

生成唯一链接分享任何对话，任何人无需登录即可访问。分享的对话是只读的，包含所有消息和文件。

**分享与导出对比：**

| 功能 | 分享 | 导出 |
|------|------|------|
| **访问方式** | 任何人通过链接 | 仅下载文件 |
| **格式** | 网页查看 | JSON/CSV/Parquet |
| **文件** | 可下载 | 不包含 |
| **更新** | 反映当前状态 | 导出时的快照 |

## 创建分享链接

点击对话头部的分享按钮创建公开链接。

**流程说明：**

| 步骤 | 操作 |
|------|------|
| **首次分享** | 系统生成唯一分享 ID 和链接 |
| **已分享** | 显示现有分享链接 |
| **链接格式** | `yourdomain.com/share/abc123xyz` |

## 访问分享对话

任何人通过链接即可查看对话，无需登录。

**访问者可见内容：**

| 功能 | 说明 |
|------|------|
| **消息** | 所有对话轮次和响应 |
| **文件** | 文件列表及下载按钮 |
| **元数据** | 对话标题、类型和时间戳 |
| **格式** | 简洁的只读界面 |

**访问者不可见内容：**

- 用户信息（所有者姓名、邮箱）
- Agent 配置详情
- 系统提示词和内部设置
- 编辑或删除功能

## 下载文件

分享的对话包含所有附加文件。

**下载选项：**

| 选项 | 使用方法 |
|------|---------|
| **单个文件** | 点击文件名旁的下载图标 |
| **所有文件** | 点击"下载全部"获取 ZIP 压缩包 |
| **文件类型** | 对话中的任何文件类型 |

## 管理分享链接

通过分享弹窗控制您的分享对话。

**可用操作：**

| 操作 | 效果 |
|------|------|
| **复制链接** | 复制分享 URL 到剪贴板 |
| **新标签打开** | 预览他人看到的效果 |
| **删除分享** | 撤销访问权限，链接失效 |

## 删除分享链接

随时删除分享链接以移除公开访问。

**删除流程：**

```
删除分享 → 链接失效 → 访问者看到 404 错误
```

对话保留在您的账户中，仅移除公开链接。

## 分享状态

从对话头部检查对话当前是否已分享。

**状态指示：**

| 状态 | 指示 |
|------|------|
| **未分享** | 显示分享按钮 |
| **已分享** | 分享按钮显示现有链接 |

## 使用场景

| 场景 | 分享如何帮助 |
|------|------------|
| **团队协作** | 与团队成员分享分析结果 |
| **文档记录** | 创建流程的公开参考 |
| **支持工单** | 与支持人员分享对话历史 |
| **教学演示** | 向学生展示工作流程 |
| **报告汇报** | 向干系人提供对话证据 |

## 隐私注意事项

**分享的内容：**

- 对话消息和轮次
- 文件内容和名称
- 对话元数据（标题、类型、时间戳）

**受保护的内容：**

- 您的账户信息
- Agent 配置
- 私有系统设置
- 其他对话

**访问控制：**

分享链接公开但不公示 - 只有拥有确切 URL 的人才能访问。搜索引擎不会索引这些页面。

## 说明

**无法编辑：** 访问者无法修改或回复分享的对话

**只读文件：** 文件可下载但无法通过分享链接编辑

**无身份验证：** 任何拥有链接的人都可访问（无密码保护）

**动态快照：** 分享显示对话当前状态，更新立即反映

**所有者权限：** 仅对话所有者可创建或删除分享链接
