# 外部 Agent API

将优秀的 Agent 和工作流以 API 形式对外提供服务，构建 Agent 调用生态。

## 当前分享方式

目前开发的 Agent 和工作流只能通过有限方式分享：

| 分享方式 | 说明 | 限制 |
|---------|------|------|
| 结果分享 | 分享对话运行结果 | 无法让他人直接使用 Agent |
| 打包开源 | 导出配置文件供社区使用 | 需要他人手动导入和配置 |

## 即将支持

Agent 开发者可以将 Agent 和工作流开放给他人调用，构建 Agent 服务市场：

### 核心能力

- **对外开放**：将您的 Agent 和工作流暴露为公开 API，供其他用户和开发者调用
- **授权管理**：控制谁可以调用您的 Agent，设置访问权限和调用配额
- **使用监控**：查看 API 被谁调用、调用次数和使用情况
- **收益管理**：为 Agent 调用设置收费，与支付协议对接

### 应用场景

| 场景 | 价值 |
|------|------|
| **Agent 服务化** | 将开发的 Agent 作为服务提供给其他开发者 |
| **商业变现** | 优秀的 Agent 可以通过调用收费获得收益 |
| **生态繁荣** | 开发者共享 Agent 能力，用户按需调用 |

## Agent 支付生态

项目将积极探索 Agent 商业化和支付生态：

### 协议探索

正在关注和研究以下 Agent 支付协议：

- **AP2**：Google 推出的 Agent 支付协议
- **ACP (Agentic Commerce Protocol)**：OpenAI 推出的 Agent 商业协议
- **ANP**：ANP 社区推动的智能体通信协议

通过支持这些协议，未来可能实现 Agent 服务的自动化交易和收益分配。
