# 未来路线图

平台持续发展，为用户带来更强大的 Agent 能力和更好的协作体验。

## 规划概述

我们的发展聚焦在三个方向：**能力增强**、**生态建设** 和 **体验优化**。

## 即将推出

以下功能即将上线或正在积极开发中：

| 功能 | 状态 | 核心价值 |
|------|------|---------|
| [多模态支持](multimodal.md) | 即将支持 | VLM 让 Agent 具备视觉理解能力 |
| [团队资源共享](resource-sharing.md) | 即将支持 | Agent、工作流、Prompt 在团队内共享 |
| [Agent Skills](skills-context.md) | 即将支持 | 渐进式上下文工程，提升效率和能力 |

## 未来规划

这些功能在持续探索和规划中：

| 功能 | 方向 | 核心价值 |
|------|------|---------|
| [外部 Agent API](external-api.md) | 规划中 | 将 Agent 开放给外部调用，构建服务生态 |
| [用户分析](analytics.md) | 规划中 | 效果评估和团队洞察 |

*我们将持续倾听用户反馈，不断完善平台功能。*
