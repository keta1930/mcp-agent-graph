# 团队资源共享

团队成员间的资源共享功能即将上线，打破当前资源完全独立的限制。

## 当前状态

目前团队成员的资源完全独立：

| 资源类型 | 当前状态 | 影响 |
|---------|---------|------|
| Agent | 仅个人可见 | 无法复用团队成员创建的 Agent |
| 工作流 | 仅个人可见 | 无法共享流程设计 |
| Prompt | 仅个人可见 | 提示词无法在团队内传播 |

## 即将支持

团队资源共享将允许成员间共享和协作：

### 核心能力

- **资源共享**：将 Agent、工作流、Prompt 等资源共享给团队成员，未来还将支持 Agent Skills
- **权限控制**：设置资源的可见范围和使用权限
- **协作编辑**：支持团队成员协作完善共享资源
- **版本管理**：追踪共享资源的修改历史

### 预期效果

共享功能将带来：

| 场景 | 价值 |
|------|------|
| **避免重复** | 一次配置，全团队使用 |
| **知识沉淀** | 优秀的 Agent 和 Prompt 在团队内传承 |
| **协作提升** | 团队成员共同完善工作流 |
| **快速上手** | 新成员直接使用团队已有资源 |

---

*资源共享将让团队协作更加高效，知识和经验得以沉淀和传承。*
