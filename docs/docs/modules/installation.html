Installation

Get started with MAG SDK by following these simple installation steps.

Requirements

Python 3.8+ is required to run MAG SDK.

  • Python 3.8 or higher
  • pip package manager
  • Internet connection (for downloading dependencies)

Installation Methods

Method 1: Install from PyPI (Recommended)

The easiest way to install MAG SDK is using pip:

pip install mag-sdk

Method 2: Install from Source

For development or the latest features, install from the source code:

# Clone the repository
git clone https://github.com/your-repo/mag.git
cd mag

# Install in development mode
pip install -e .

Verify Installation

After installation, verify that MAG SDK is working correctly:

import mag
print(f"MAG SDK version: {mag.__version__}")
print("Installation successful!")

If no errors occur, you're ready to start using MAG SDK!

Initial Configuration

MAG SDK will automatically start the backend service when you first use it. You can also start it manually:

import mag

# Start the MAG service
mag.start()

# Check if service is running
if mag.is_running():
    print("MAG service is running!")
else:
    print("Failed to start MAG service")

Troubleshooting

Permission denied error

If you encounter permission errors, try installing with --user flag:

pip install --user mag-sdk

Python version incompatibility

Make sure you're using Python 3.8+:

python --version

Service startup issues

If the MAG service fails to start, check if the default port is available or try restarting:

mag.stop() # Stop existing service
mag.start() # Start fresh
