Quick Start

Get up and running with MAG SDK in minutes. This guide will walk you through creating and executing your first graph.

1 Import and Initialize

First, import MAG SDK and ensure the service is running:

import mag

# The service will start automatically when first used
print(f"MAG SDK version: {mag.__version__}")

2 Explore Available Graphs

Check what graphs are available in your system:

# List all available graphs
import mag

graphs = mag.list_graph()
print("Available graphs:")
for graph in graphs:
    print(f"  - {graph}")

If you don't have any graphs yet, you can create one using the AI generation feature or import existing graphs.

3 Execute a Graph

Now let's execute a graph. MAG SDK offers multiple execution modes:

Basic Execution (Wait for completion) 

# Execute graph and wait for completion
import mag

result = mag.run_graph(
    name="your_graph_name",
    input_text="Your input here",
    stream=False,      # Non-streaming mode
    background=False   # Wait for completion
)

print(f"Conversation ID: {result.get('_id')}")
print(f"Final Result: {result.get('final_result')}")

Background Execution (Non-blocking)

For long-running graphs, use background execution:

# Start graph execution in background
conversation_id = mag.run_graph(
    name="your_graph_name",
    input_text="Your input here",
    background=True    # Returns immediately with conversation_id
)

print(f"Graph started in background: {conversation_id}")
print("You can continue with other tasks while it runs...")

# Later, check the results
result = mag.get_conversation_detail(conversation_id)
if result.get("completed"):
    print(f"Graph completed! Result: {result.get('final_result')}")
else:
    print("Graph is still running...")

Streaming Execution (Real-time updates)

Get real-time updates during graph execution:

# Execute graph with streaming updates
response = mag.run_graph(
    name="your_graph_name",
    input_text="Your input here",
    stream=True,       # Enable streaming
    background=False
)

# Process streaming updates
for chunk in response:
    if chunk.get("type") == "conversation_created":
        print(f"Conversation started: {chunk.get('conversation_id')}")

    elif chunk.get("type") == "node_start":
        print(f"Starting node: {chunk.get('node_name')}")

    elif chunk.get("choices"):
        # AI response content
        delta = chunk["choices"][0].get("delta", {})
        if delta.get("content"):
            print(delta["content"], end="", flush=True)

    elif chunk.get("type") == "graph_complete":
        print("\nGraph execution completed!")

4 Working with Results

Understanding the structure of graph execution results:

# Get conversation details
conversation = mag.get_conversation_detail("conversation_id_here")

# Key fields in the result:
print(f"Conversation ID: {conversation.get('_id')}")
print(f"Graph Name: {conversation.get('graph_name')}")
print(f"Status: {'Completed' if conversation.get('completed') else 'Running'}")
print(f"Final Result: {conversation.get('final_result')}")
print(f"Number of rounds: {len(conversation.get('rounds', []))}")
print(f"Start time: {conversation.get('start_time')}")

# Access individual rounds
for i, round_data in enumerate(conversation.get("rounds", [])):
    print(f"Round {i+1}: {round_data.get('node_name')}")
    # Access messages in this round
    for message in round_data.get("messages", []):
        print(f"  {message.get('role')}: {message.get('content')[:100]}...")

Common Patterns

Check and Wait for Completion 

import time

# Start background execution
conversation_id = mag.run_graph(
    name="my_graph",
    input_text="input",
    background=True
)

# Poll for completion
while True:
    result = mag.get_conversation_detail(conversation_id)
    if result.get("completed"):
        print("Completed!")
        break
    time.sleep(5)  # Wait 5 seconds

Batch Processing 

# Process multiple inputs
inputs = ["input1", "input2", "input3"]
conversation_ids = []

for input_text in inputs:
    conv_id = mag.run_graph(
        name="processor",
        input_text=input_text,
        background=True
    )
    conversation_ids.append(conv_id)

print(f"Started {len(conversation_ids)} jobs")
