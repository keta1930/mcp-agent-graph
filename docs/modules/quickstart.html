快速开始

MAG SDK 是一个强大的智能体开发框架，本指南将帮助您快速上手使用。

安装部署

1. 安装SDK

Shell 
pip install mag-sdk

2. 导入SDK

Python 
import mag

3. 启动服务

Python 
# 启动MAG服务器
mag.start()

# 检查服务状态
if mag.is_running():
    print("MAG服务已启动")

服务管理

基本服务控制

Python 
import mag

# 启动服务
success = mag.start()
if success:
    print("服务启动成功")

# 检查服务状态
running = mag.is_running()
print(f"服务运行状态: {running}")

# 关闭服务
mag.shutdown()
print("服务已关闭")

注意：在使用任何SDK功能之前，必须先调用 mag.start() 启动服务。

第一个例子

以下是一个简单的例子，展示如何使用MAG SDK进行基本的对话：

Python 
import mag

# 启动服务
mag.start()

# 添加一个模型（如果还没有）
model_config = {
    "name": "gpt-4",
    "base_url": "https://api.openai.com/v1",
    "api_key": "your-api-key",
    "model": "gpt-4"
}
mag.add_model(model_config)

# 发起对话
response = mag.chat_completions(
    user_prompt="你好，请介绍一下MAG SDK",
    model="gpt-4"
)

print(response)
